# VISH 0.1.0 Alpha 4

This alpha fixes the update flow.

- Sparkle builds now ship with a real feed URL and EdDSA public key.
- The app uses Sparkle for update checks when configured.
- Local or custom builds without Sparkle config now open GitHub Releases instead of showing developer environment variables.
- Release packaging now generates a signed `appcast.xml` with monotonic Sparkle build versions.

Still alpha: the app is locally signed and not notarized yet.
