# VISH 0.1.0 Alpha 5

This alpha adds richer local controls for clipboard, privacy, and file indexing.

- Clipboard History v2 adds pinned entries, retention controls, source-app metadata, ignored source apps, edit/delete/clear controls, and Tab actions for Edit & Paste, Pin / Unpin, and Delete.
- Settings now includes a Privacy dashboard for Accessibility, Full Disk visibility, Clipboard state, Local AI state, and local Application Support data.
- Full Disk file search now has VISH-level folder exclusions for common folders plus custom excluded folders.
- Folder exclusions apply to Spotlight results, fallback filename indexing, semantic/vector search, vector cache pruning, and file watcher paths.
- Release metadata now uses Sparkle build version 5.

Still alpha: the app is locally signed and not notarized yet.
