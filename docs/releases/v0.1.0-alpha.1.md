# VISH 0.1.0 Alpha 1

First public alpha DMG for VISH, a native macOS launcher built around speed, local-first workflows, and explicit local AI.

## Install

1. Download `vish-0.1.0-alpha.1.dmg` from this release.
2. Open the DMG.
3. Drag `vish.app` to Applications.
4. Launch VISH and use `Option-Space`.

This alpha DMG is not notarized yet, so macOS may require opening it through System Settings > Privacy & Security or Control-click > Open.

## Included

- Native AppKit launcher panel with frosted compact UI.
- App search, system actions, calculator, URLs, web search, Quicklinks, snippets, clipboard history, and file search.
- Full Disk Access flow and indexed file search with progress.
- Rich result previews, Quick Look, Tab-driven universal actions, file buffering, and result source chips.
- Optional local AI through Ollama with inline answers.
- Selected-result AI actions for Ask, Summarize, and Explain.
- Explicit semantic file finder through `ai find ...` using local file indexing and embeddings when configured.
- Polished Settings Setup pane with readiness cards for Launcher, Files, Clipboard, and AI.
- First-run onboarding with permissions and feature-default flow.

## Benchmarks

- Launch to process: `107 ms`
- Idle RSS: `75.3 MB`

Known measurement gap: full Instruments signpost traces are still needed for hotkey-to-frame, keystroke-to-render, and Spotlight p95 before claiming production 1.0 readiness.
